## 実行

以下の内容で`.env.local`を作成

```
SERVICE_DOMAIN=<microCMSのサービスドメイン>
API_KEY=<microCMSのAPIキー>
```

その後、以下のコマンドで実行

```
yarn build & yarn start
```
